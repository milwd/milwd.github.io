About Me

I am a MSc student of Robotics Engineering at Università degli Studi di Genova. Having received full undergraduate scholarship, I did my bachelors in Electrical Engineering at Shahid Beheshti University in Iran. Besides a student, I have been a researcher, a teaching assistant, a team member, and a national committee member of RoboCup IranOpen.

I had the privilege of establishing the Vision division of SBU’s Robotics and Intelligent Automation Lab, and have been rewarded by the university president and department dean for my contributions to many First place victories of the team in international competitions. I am extremely enthusiastic about my graduate studies.

Projects

1:10 Scale Self-Driving Robot Car
Leading the evolution of lane detection methodologies from filter-based techniques with novel density-based trackers to deep learning models, achieving robust path planning in collaboration with the Control section. Train, test, and deployment of various object detection models, enabling the robot to accurately identify and respond to traffic signs and obstacles. Development of a synthetic data generation and augmentation pipeline which helped improve the robustness of the model significantly. Youtube

Face and Hand Gesture Detection
With a focus on detecting sign language, this computer vision project uses pretrained optimized neural networks from Google’s Mediapipe to get hand landmarks which are then passed into a NN model for gesture recognition (Acc 96%). The result is then sent to an ESP32 board. Code

Image Recovery with Genetic Algorithm
A slightly altered Genetic search algorithm is implemented to recover the famous Alan Turing image from random pixels. The project includes selection, simultaneous crossover, and slightly-enhanced adaptive mutation, to get results. Code

Sobel Edge Detection in VHDL
Sobel edge detection algorithm implementation in VHDL for FPGA boards. This simulation project was programmed in Xilinx ISE. The image pixels and file handling were coded in Python. Code

Movie Recommendation System
This system recommends movies to users based on their ratings and (cosine) similarities to other users. It incorporates a dynamic dataset of movies and user ratings. The CLI app was programmed in C++. Code

GAN / Conditional GAN for MNIST Dataset
Generation of synthetic samples for MNIST image dataset with the help of Generative Adversarial Networks. Conditional generation is also implemented based on CGAN architecture.

Ball Balancer Bot
A motor tries to balance a ball on a rod (plate). By implementing PID controllers and using ToF Laser-ranging sensors, the device can hold the ball stable on any distances on the rod, specified by user. It uses ESP8266 as controller device and server for web connection.
